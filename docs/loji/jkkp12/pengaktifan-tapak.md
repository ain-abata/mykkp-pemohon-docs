<div className="step-card">
  ### Langkah 2: Masuk Ke Halaman Profil Tempat Kerja

import imgProfilTK from '@site/assets/images/loji/jkkp12/halaman_profil_tk.png';

<ExpandableImage
    src={imgProfilTK}
    alt="Halaman Profil Tempat Kerja"
    caption="Halaman Profil Tempat Kerja"
    width={450} />

  1. Klik butang butang **Profil Tempat Kerja** dan seterusnya klik pada butang **Pengaktifan Tempat Kerja**.
  2. Aktifkan tempat kerja **Kontraktor Prinsipal Kerja Pembinaan** atau **Kontraktor Kerja Pembinana Tunggal** yang dilantik oleh Klien/Pemberitahu dengan klik pada butang **Aktif**.
</div>

<div className="step-card">
  ### Langkah 3: Masuk ke Halaman Pemberitahuan Tapak Pembinaan

import imgJKKP12 from '@site/assets/images/loji/jkkp12/halaman_jkkp12.png';

<ExpandableImage
  src={imgJKKP12}
  alt="Halaman JKKP12"
  caption="Halaman JKKP12"
  width={450} />

  1. Klik pada butang **JKKP12** pada menu sebelah kiri.
  2. Klik pada butang **Senarai Tapak Pembinaan** untuk memaparkan halaman **Penguursan JKKP 12 Senarai Tapak Pembinaan**.
  3. Klik pada butang **Pengaktifan Tapak Pembinaan**.
  4. Klik pada butang **Tindakan**. Klik pada butang **Aktifkan Tapak** untuk mengaktifkan tapak pembinaan.
  <Admonition type="info">
  Paparan **Pemberitahuan Tapak Pembinaan** akan terpapar dalam tab baru.
  </Admonition>
</div>

<div className="step-card">
  ### Langkah 4: Pengaktifan Tapak Pembinaan

  1. Semak maklumat **Pemberitahuan Tapak Pembinaan**.
  2. Untuk mengaktifkan tapak pembinaan, klik pada butang **Aktifkan Tapak**.
  3. Klik pada butang **Ya** untuk meneruskan tindakan pengaktifan tapak atau **Tutup** untuk batal tindakan pengaktifan tapak.
  4. Pop-up Tapak Pembinaan berjaya diaktifkan akan terpapar.
  5. Klik pada butang **Lihat Sijil Pemberitahuan Kerja Pembinaan** untuk memaparkan Sijil Pemberitahuan Kerja Pembinaan.
  6. Klik butang **OK** untuk memaparkan halaman Ruangan-Kerja Tapak Pembinaan.
</div>
